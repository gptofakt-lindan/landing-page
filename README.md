# Landing Page Project

A responsive landing page built as part of **The Odin Project Foundations Course**.

## Preview

This project is a simple landing page focused on learning and self-improvement. It includes:

- Navigation header
- Hero section
- Information cards
- Inspirational quote section
- Call-to-action section
- Footer

## Built With

- HTML5
- CSS3
- Flexbox

## Concepts Practiced

This project helped me strengthen my understanding of:

### HTML

- Semantic page structure
- Links and images
- Accessibility using alt text
- Organizing content into sections

### CSS

- Selectors and properties
- Typography and spacing
- Colors and backgrounds
- Borders and border-radius
- Responsive layouts

### Flexbox

- `display: flex`
- `flex-direction`
- `justify-content`
- `align-items`
- `flex-wrap`
- Flexible sizing with `flex`

## What I Learned

While building this project, I practiced:

- Introduction to CSS
- Cascading rule and specificty
- About box model and its properties like margin, border and padding
- Box sizing
- Block and inline
- Introduction to Flexbox and flexbox container and flexbox items
- About flexbox growing and shrinking
- About flexbox main and cross axes
- How justify-content and align-items behaves/alligns as per main and cross axis

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/gptofakt-lindan/landing-page.git
```

2. Open the project folder

```bash
cd landing-page
```

3. Open `index.html` in your browser
